Welcome! University Library

Sejong University Library was completed in may 1998 and provides students with a space for learning

This building is a library of Sejong University.
Another name for the building is Dongcheongwan, Sometimes you can see Dongcheongwan in building list at homepage.
Most of all knows that there are only 9 floors above the ground, because the elevator is only going to the 9th floor.
But in fact, there are the 10th floor and the second basement floor.
Each floor has its own characteristics.
On the second basement floor and the tenth floor above the ground have preservation documents.
On the first to third floors of the basement, there are many reading rooms.
On the 8th floor, there is a studio that takes internet lectures.
On the fourth floor, there are a creative discussion lounge and an infinite imagination space, which is blocked for whatever reason.
Also, there are many study rooms on 4th, 7th and 1st basement level, so you can use them if you make a reservation on the Internet homepage.
When the first semester begins in the library, there is an event for first graders every time. If you are a freshman, you should participate.
